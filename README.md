"# Kwiko Portfolio"

"## Overview"

"Kwiko Technologies And Services Private Limited single-page Next.js portfolio featuring a premium dark glassmorphism-inspired UI with smooth motion, SEO metadata, and clear calls to action."

"## Features"

"- Hero with animated typography, logo emphasis, and CTA buttons"
"- Services, mission, why-choose-us, projects, and tech stack showcases"
"- Glass cards, floating particles, animated stats, sticky navbar, and WhatsApp CTA"
"- Responsive layout, SEO metadata, and gradient/glow styling throughout"

"## Tech Stack"

"Next.js 15 · Tailwind CSS · Framer Motion · Lucide React icons"

"## Getting Started"

"```bash"
"npm install"
"npm run dev"
"```"
